# Number Baseball (숫자 야구)

React + TypeScript + Vite + Ant Design + SCSS로 만든 프론트엔드 전용 숫자 야구 게임입니다. 점수는 `localStorage`(`numberBaseballScore`)에 누적 저장됩니다.

## 필요 환경

- Node.js 18+ 권장
- npm

## 설치

```bash
npm install
```

## 실행

개발 서버:

```bash
npm run dev
```

프로덕션 빌드:

```bash
npm run build
```

빌드 결과 미리보기:

```bash
npm run preview
```

## 주요 패키지

- `react`, `react-dom`
- `react-router-dom`
- `antd`
- `sass`
- `vite`, `typescript`, `@vitejs/plugin-react`
